Target Relevance — Theme / Cluster 근거

E/I Balance와 Neuroimmune은 Target Relevance를 판단할 때 쓰는 SKBP 내부 biology map입니다. asset의 target과 MoA가 아래 theme/cluster 중 어디에 들어가는지 먼저 확인하고, 맞지 않으면 Target Relevance를 높게 주지 않습니다.

E/I Balance

balance between excitation and inhibition

  • Ion Channel
  • Inhibitory Tone 강화
  • Synaptic Transmission
  • Chloride Homeostasis
  • Network Modulation

Neuroimmune

neuroimmune biology across CNS disease

  • CNS 손상 면역반응
  • 교세포 향상성
  • Cytokine 신경조절
  • 손상/질환 면역조절
  • 말초 면역기관 연결

Parameter Guide

각 항목에서 실제로 보는 핵심 쟁점

1. Target Relevance

Target, disease biology, indication이 SKBP의 E/I Balance 또는 Neuroimmune theme/cluster에 직접 맞는지 봅니다. 단순 CNS 질환보다 target-level fit이 중요합니다.

E/I Balance Ion Channel, Inhibitory Tone, Synaptic Transmission, Chloride Homeostasis, Network Modulation
Neuroimmune Glial homeostasis, CNS injury immune response, Cytokine neuromodulation, disease immune modulation

2. Competitive Landscape

같은 indication, target, MoA에서 앞선 competitor가 얼마나 강한지와 차별 가능한 위치인지를 봅니다. 유사 pipeline은 similar_pipeline_count, high/medium/low similarity, matched dimensions, shared data points, differentiating data points, evidence source를 함께 기록합니다.

같은 indication, target, MoA에서 앞선 competitor가 얼마나 강한지 봅니다. 경쟁이 없다는 것보다 차별 가능한 위치인지가 핵심입니다.

3. MoA Validity

MoA가 질환 biology와 기능적으로 연결되는지 봅니다. claim만 있는지, class validation 또는 asset-specific validation이 있는지가 점수 차이를 만듭니다.

4. Platform Attractiveness

modality와 platform이 SKBP 관점에서 다루기 좋은지, 반복 가능성과 차별성이 있는지 봅니다. small molecule, ASO, siRNA는 기본 선호도가 높습니다.

5. Expansion Potential

같은 biology, target, platform으로 adjacent indication이나 후속 pipeline 확장이 가능한지 봅니다. 단순 label 확장이 아니라 실제 biology 근거가 필요합니다.

6. Data Maturity

현재 stage에 맞는 asset-specific data가 보이는지 봅니다. preclinical이면 실험 readout, clinical이면 human safety/efficacy/biomarker 근거가 핵심입니다.

7. Marketability

Marketability는 환자 풀에서 시작해 실제 매출 가능성까지 단계적으로 좁혀 봅니다. A는 치료 가능한 환자 수, B는 경쟁 리스크를 반영하기 전 peak sales, C는 경쟁·가격·확장성을 반영한 obtainable peak sales입니다.

A. TAP Total Patient Pool x Diagnosis Rate x Eligibility Rate x Treatable Subgroup Rate
B. Unrisked Peak Sales TAP x Annual Net Price x Peak Penetration x Treatment Duration Factor
C. Obtainable Peak Sales Unrisked Peak Sales x Competition Haircut x Pricing Power Adjustment x Expansion Capacity Adjustment

Sales output unitmillion USD

Target Relevance

Target을 먼저 identify한 뒤 Theme과 Cluster를 분리해서 기록합니다. Target 아래에는 Theme: ___ (Cluster: ___) 형식으로 표시합니다.

Competitive Landscape

FIC만 높게 보지 않고, 이미 1개 정도 앞서가는 약물이 있는 경우를 가장 선호합니다. 판단 불가이면 0점으로 둡니다.

MoA Validity

왜 작동할 것 같은지의 설득력입니다. 2점 이상은 PMID, 저널, conference abstract, company deck 같은 출처를 반드시 남깁니다.

Data Maturity

clinical data는 제외하고 preclinical 실험값만 봅니다. MoA는 기전의 설득력, Data는 실제 실험 결과의 성숙도입니다.

Platform Attractiveness

주요 관심 모달리티는 small molecule, ASO, siRNA입니다. AOC, antibody 등은 둘째 관심 모달리티로 평가합니다.

Marketability

A. TAP를 먼저 추정하고, B. Unrisked Peak Sales를 계산한 뒤, C. Obtainable Peak Sales로 현실화합니다. 모든 sales 결과는 million USD 단위로 기록합니다.

근거 기록 규칙

  • Rubric기준 설명은 rubric 섹션에만 남깁니다.
  • main_line_summary각 scoring 항목에서 왜 그 점수가 나왔는지만 한 줄로 기록합니다.
  • evidence_sources점수 판단에 사용한 출처 URL과 핵심 근거를 분리합니다.
  • investigation_note조사 과정, 확인한 항목, 판단 경계를 짧게 남깁니다.
  • uncertain_points자료 부족, 모호한 claim, 추가 확인 필요사항을 따로 기록합니다.

경쟁사 유사도 기록

경쟁 pipeline은 similar_pipeline_count, high/medium/low similarity 개수, 각 유사 pipeline의 matched dimensions, shared data points, differentiating data points, evidence source를 함께 기록합니다.
